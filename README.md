# Silly-Blog

基于Flask和Vue.js的博客系统.

* 使用Flask编写的Web APP仅提供RESTful API
* 使用Vue.js编写的前端SPA

## 当前进度

后台管理只完成了登录、home首页、用户管理、标签管理、分类管理和文章管理等功能:
